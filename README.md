# cardano-tx-tools

Cardano transaction tooling: builder, structural diff, blueprint
decoding. Uses [`cardano-node-clients`][cnc] for Provider/N2C access but
is not a node client.

Bootstrapping in progress. The actual code is moving from
[lambdasistemi/cardano-node-clients#152][issue].
